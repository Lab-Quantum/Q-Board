## Installation

> All the `code` required to get started

### Clone

> Clone this repo to your local machine using `https://github.com/Lab-Quantum/Q-Board.git`

```shell
$ git clone https://github.com/Lab-Quantum/Q-Board.git
```

### Setup

> Install all package first

```shell
$ make install
```

> Run

```shell
$ make run
```
